## What Aether Is

**Aether is a Claude-native system** - You interact with it by typing commands directly in Claude (like `/ant:init "Build a REST API"`), NOT by running Python scripts.

Think of it like CDS - you type `/cds:plan-phase` and it works. Same with Aether:
- `/ant:init "goal"` - Initialize a new project
- `/ant:status` - Show colony status
- `/ant:execute 1` - Execute Phase 1
- `/ant:focus "security"` - Guide colony attention

## Key Differences from Python Script

| Aspect | Old (Python) | New (Claude-Native) |
|--------|--------------|---------------------|
| Invocation | `python3 .aether/demo.py` | `/ant:init "goal"` |
| State | In-memory objects | JSON files |
| Execution | Sequential | Emergent with phases |
| Visibility | Limited | Checkpoints at phases |
| Integration | Separate tool | Part of Claude workflow |
