## Data Sources (EXISTING - read only)

1. **QUEEN.md** (.aether/docs/QUEEN.md)
   - Contains wisdom entries in 5 categories (philosophies, patterns, redirects, stack, decrees)
   - Has METADATA block at bottom with stats JSON
   - Used for: wisdom count metric

2. **learning-observations.json** (.aether/data/learning-observations.json)
   - Structure: {"observations": [{"content_hash", "content", "wisdom_type", "observation_count", "colonies_contributed[]", "first_observed", "last_observed"}]}
   - Used for: pending promotions (observations meeting threshold but not promoted)

3. **learning-deferred.json** (.aether/data/learning-deferred.json)
   - Structure: {"deferred": [{"content_hash", "content", "wisdom_type", "observation_count", "deferred_at"}]}
   - Used for: pending promotions (deferred proposals)

4. **midden.json** (.aether/data/midden/midden.json)
   - Structure: {"signals": [{"type", "content", "source", "created_at", "context"}]}
   - Type "failure" entries used for: recent failures metric

5. **COLONY_STATE.json** (.aether/data/COLONY_STATE.json)
   - Used for: last activity timestamps, current phase info

## Existing Functions to Reference

- learning-check-promotion: Returns proposals meeting thresholds
