Fix two issues in syncAetherToRepo() method (lines 667-767):

**Bug 1 - Counter outside dry-run block:**
Current code (lines 698-724) has `copied++` at line 723 OUTSIDE the if (!dryRun) block. This causes dry-run to report all files as copied even when nothing was written.

Fix: Move the counter increment to only count files that would be (or are) actually copied:

```javascript
if (!dryRun) {
  fs.mkdirSync(path.dirname(destPath), { recursive: true });

  // Hash comparison
  let shouldCopy = true;
  if (fs.existsSync(destPath)) {
    const srcHash = this.hashFileSync(srcPath);
    const destHash = this.hashFileSync(destPath);
    if (srcHash === destHash) {
      shouldCopy = false;
      skipped++;
    }
  }

  if (shouldCopy) {
    this.copyFileAtomic(srcPath, destPath);  // Use atomic pattern
    copied++;  // FIX: Only increment when actually copied
  }
} else {
  // dry-run: count files that WOULD be copied (after hash check logic)
  let wouldCopy = true;
  if (fs.existsSync(destPath)) {
    const srcHash = this.hashFileSync(srcPath);
    const destHash = this.hashFileSync(destPath);
    if (srcHash === destHash) {
      wouldCopy = false;
    }
  }
  if (wouldCopy) {
    copied++;  // Count only files that would actually be copied
  }
}
```
