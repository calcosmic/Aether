# v1.4 Requirements: Deep Cleanup & Simplification

**Created:** 2026-02-20
**Updated:** 2026-02-20 (after architect review)
**Goal:** Remove dead weight, consolidate duplicates, simplify structure — make the repo understandable at a glance.

## The Problem

The Aether repo has grown organically over 25 phases. It now contains:
- Dead duplicates (`.aether/agents/`, `.aether/commands/`)
- Orphaned worktree salvage files
- Completed planning phases spread across `.planning/phases/` and `docs/plans/`
- Unclear what is "core" vs "archived"
- Bash line wrapping bug causing command execution errors

This makes it hard to:
- Find what you need
- Understand what's actually being distributed
- Know what's important vs dead weight

## Requirements
