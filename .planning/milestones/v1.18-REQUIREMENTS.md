# Requirements: Aether v1.18 (Archived)

**Defined:** 2026-05-14
**Shipped:** 2026-05-14
**Core Value:** Aether should feel alive and truthful at runtime, not only look clever in wrappers or tests.

## v1.18 Requirements: Hybrid Runtime Parity & Release Gate

### TS Host Reliability (REL)

- [x] **REL-01:** `npm --prefix .aether/ts-host run typecheck` passes with zero errors
- [x] **REL-02:** Full TS test suite (`npm --prefix .aether/ts-host test`) exits cleanly without hangs
- [x] **REL-03:** Event bridge teardown awaits full subprocess/readline cleanup
- [x] **REL-04:** Completion file paths are unique per lifecycle run (no fixed `/tmp/aether-lifecycle/`)
- [x] **REL-05:** Individual tests and suite mode both pass for lifecycle and golden workflow

### Platform Dispatch Correctness (DSP)

- [x] **DSP-01:** Codex dispatch passes the worker prompt to `codex exec` (not just `--output-schema`)
- [x] **DSP-02:** Claude argument construction is tested and verified
- [x] **DSP-03:** OpenCode argument construction is tested and verified
- [x] **DSP-04:** Simulation fallback is explicit and does not mask broken real dispatch
- [x] **DSP-05:** Spawn-log/spawn-complete only records manifest workers

### Go Runtime Test Restoration (GOT)

- [x] **GOT-01:** `go test ./cmd` passes with zero failures
- [x] **GOT-02:** Resume dashboard signal injection failure is resolved
- [x] **GOT-03:** Workspace/planning cleanup state is resolved (deleted files committed or archived)
- [x] **GOT-04:** Scratch files (`seal-debug.ts`, etc.) are removed or promoted
- [x] **GOT-05:** Go remains sole authority for `.aether/data` mutation

### Classic Parity Coverage (PAR)

- [x] **PAR-01:** Golden tests verify build ceremony matches v5.4 baseline
- [x] **PAR-02:** Golden tests verify continue ceremony matches v5.4 baseline
- [x] **PAR-03:** Oracle confidence loop behavior is tested against v5.4 baseline
- [x] **PAR-04:** Swarm/dashboard visibility is tested against v5.4 baseline
- [x] **PAR-05:** Install/update flow is tested against v5.4 baseline
- [x] **PAR-06:** State mutation through approved APIs is tested against v5.4 baseline
- [x] **PAR-07:** Any Classic behavior intentionally not restored is documented

### Release Gate (REL)

- [x] **REL-06:** Dev channel publish succeeds
- [x] **REL-07:** Downstream smoke test passes: `aether update --force`, `aether init`, `aether plan`, `aether build 1`, `aether continue`, `aether oracle`
- [x] **REL-08:** Exact blocker list is recorded before stable release
