## Central Architecture

colony-prime (aether-utils.sh) serves as the hub for all five pipelines:

```
continue-advance.md          colony-prime (hub)           build-wave.md
  instinct-create ------>  INSTINCTS section  --------->  { prompt_section }
  memory.phase_learnings -> PHASE LEARNINGS  --------->  { prompt_section }
  pheromone-write -------->  ACTIVE SIGNALS  --------->  { prompt_section }

continue-finalize.md
  learning-promote-auto --> QUEEN WISDOM  ------------->  { prompt_section }

CONTEXT.md -----------------> KEY DECISIONS  ---------->  { prompt_section }
flags.json -----------------> BLOCKER WARNINGS  ------->  { prompt_section }

seal.md
  learning-promote-auto --> QUEEN.md (persistent wisdom)
```
