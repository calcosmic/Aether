# Phase 123 Plan 01: Dev Publish + Downstream Smoke

## Goal
Publish v1.0.38 to the dev channel and verify the full colony lifecycle in a clean downstream repo.

## Tasks

### Task 1: Publish to dev channel
**Command:**
```bash
aether publish --channel dev --binary-dest "$HOME/.local/bin"
```

**Verify:**
- Binary builds successfully
- Hub version updates to 1.0.38
- `aether-dev version` returns 1.0.38
- `aether-dev version --check` returns exit 0

**Evidence of done:**
- Publish command completes without errors
- Version check passes

### Task 2: Select and prepare downstream repo
**Target:** `~/repos/Formica`

1. Check git status is clean
2. If dirty, abort and select alternative repo
3. Record repo state before test

**Evidence of done:**
- `git status` shows clean working tree

### Task 3: Run downstream update
**Command:**
```bash
cd ~/repos/Formica
aether-dev update --force
```

**Verify:**
- Update completes without errors
- Companion files are copied (non-zero counts)
- No stale publish warnings

**Evidence of done:**
- Update output shows files copied
- Exit code 0

### Task 4: Smoke test lifecycle commands
**Commands:**
```bash
cd ~/repos/Formica
aether-dev init "Smoke test colony"
aether-dev plan
aether-dev build 1
aether-dev continue
aether-dev oracle --help  # Verify command exists
```

**Verify:**
- Each command returns exit code 0
- No panics or fatal errors
- Visual output renders (or JSON output is valid)

**Evidence of done:**
- All 5+ commands execute successfully
- Output is visible and non-empty

### Task 5: Record blocker list
1. Document any failures from Tasks 1-4
2. If no failures, record "No blockers — ready for stable release"
3. Update `.planning/STATE.md` with blocker list

**Evidence of done:**
- Blocker list written to STATE.md or separate file

### Task 6: Milestone completion
1. Mark v1.18 as complete in STATE.md
2. Update progress to 100%
3. Archive milestone if appropriate

**Evidence of done:**
- STATE.md shows 100% complete
- Milestone status is "completed"

## Verification Steps

- [ ] Task 1: Dev publish succeeds, version 1.0.38
- [ ] Task 2: Downstream repo is clean
- [ ] Task 3: Downstream update pulls latest files
- [ ] Task 4: Full lifecycle smoke test passes
- [ ] Task 5: Blocker list recorded
- [ ] Task 6: Milestone marked complete

## Rollback
- Dev channel can be republished from any previous commit
- Downstream repo colony state can be removed with `rm -rf .aether/data/`
- No irreversible changes
