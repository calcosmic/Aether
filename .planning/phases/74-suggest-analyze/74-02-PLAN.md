Create a new file in package `cmd`. This replaces the stub in compatibility_cmds.go.

```go
package cmd

import (
    "encoding/json"
```

Command structure:
- Use: "suggest-approve"
- Short: "Review and approve pheromone suggestions from suggest-analyze"
- Args: cobra.NoArgs
- Flags:
  - --dry-run (bool, default false): preview without persisting
  - --approve (string): approve a specific suggestion by ID
  - --dismiss (string): dismiss a specific suggestion by ID
  - --dismiss-all (bool): dismiss all pending suggestions

Command logic (in RunE):
1. Store nil guard
2. Load active colony state via loadActiveColonyState(). If error, return ok:true with empty suggestions (non-blocking)
3. If cs.PendingSuggestions is nil, return ok:true with empty suggestions array

4. **No flags (list mode):**
   - Filter pending suggestions: exclude dismissed ones
   - Return ok:true with suggestions array and count
   - Format each suggestion with ID, Type, Content, Reason for display

5. **--approve <id>:**
   - Find the suggestion with matching ID in cs.PendingSuggestions
   - If not found, return ok:true with not_found=true and message
   - If found and dry-run, return ok:true with would_approve=true and the suggestion details
   - If found and not dry-run:
     a. Sanitize suggestion content via colony.SanitizeSignalContent(suggestion.Content)
     b. Build content JSON: `{"text": sanitized_content}`
     c. Compute content hash: "sha256:" + sha256Sum(suggestion.Content)
     d. Determine priority from type: REDIRECT -> "high", FOCUS -> "normal", FEEDBACK -> "low"
     e. Create PheromoneSignal struct (same pattern as pheromone_write.go):
        ```go
        signal := colony.PheromoneSignal{
            ID:          generateSignalID(),
            Type:        suggestion.Type,
            Content:     json.RawMessage(contentJSON),
            Priority:    priority,
            Source:      strPtr("aether-suggest"),
            CreatedAt:   time.Now().UTC().Format(time.RFC3339),
            Active:      true,
            Strength:    ptrFloat64(0.7),
            ContentHash: &contentHash,
            Tags:        make([]colony.PheromoneTag, 0),
        }
        ```
     f. Load pheromones.json, append signal, run dedup check (same as pheromone_write.go lines 157-181), save pheromones.json
     g. Remove the suggestion from cs.PendingSuggestions (or mark it approved)
     h. Save COLONY_STATE.json
     i. Emit lifecycle event (optional, follow pheromone_write pattern)
     j. Return ok:true with approved=true and signal details

6. **--dismiss <id>:**
   - Find the suggestion with matching ID
   - If not found, return ok:true with not_found=true
   - If found and not dry-run: set suggestion.Dismissed = true, save COLONY_STATE.json
   - Return ok:true with dismissed=true

7. **--dismiss-all:**
   - Set Dismissed=true on all pending suggestions
   - Save COLONY_STATE.json
   - Return ok:true with dismissed_count

Helper functions needed (if not already in package):
```go
func strPtr(s string) *string { return &s }
func ptrFloat64(f float64) *float64 { return &f }
```
