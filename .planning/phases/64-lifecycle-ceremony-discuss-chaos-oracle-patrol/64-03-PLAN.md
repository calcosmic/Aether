Per D-08 and D-09: Create a new `patrol-check` cobra subcommand in cmd/patrol_check.go.

Register `patrolCheckCmd` with Use: "patrol-check", Short: "Run colony health checks (JSON validity, stale signals, interrupted builds)", Args: cobra.NoArgs.
Register in init() with rootCmd.AddCommand(patrolCheckCmd).

The subcommand runs 3 checks:

**Check 1: JSON Validity**
For each of the 3 files (COLONY_STATE.json, pheromones.json, session.json):
- Use os.ReadFile to load from store.BasePath()
- If os.IsNotExist: status="missing", severity="info"
- If len(data)==0: status="empty", severity="info"
- If !json.Valid(data): status="error", severity="warning", details="invalid JSON"
- Otherwise: status="healthy", severity="" (include file size in details)

**Check 2: Stale Pheromone Detection**
- Load COLONY_STATE.json to get current_phase
- Load pheromones.json to get signals
- For each active signal:
  - If sig.SourcePhase != nil and *sig.SourcePhase < state.CurrentPhase: stale (reason="references completed phase {sig.SourcePhase}, current is {state.CurrentPhase}")
  - If sig.Strength != nil and *sig.Strength <= 0: stale (reason="zero strength")
- Status: "healthy" if no stale signals, "warning" if stale found
- Details include list of stale signal IDs, types, and reasons

**Check 3: Interrupted Build Detection**
- Check for files matching patterns in store.BasePath():
  - build_manifest_*.json
  - dispatch_manifest_*.json
  - spawn-tree*.json
- Use filepath.Glob on store.BasePath() with each pattern
- If any files found: status="warning", details=list of found files
- If none found: status="healthy"

**Output structure (per D-09):**
```json
{
  "checks": [
    {
      "name": "json_validity",
      "status": "healthy",
      "files": [
        {"path": "COLONY_STATE.json", "status": "healthy", "size": 1234},
        {"path": "pheromones.json", "status": "healthy", "size": 567},
        {"path": "session.json", "status": "missing"}
      ]
    },
    {
      "name": "stale_pheromones",
      "status": "healthy",
      "stale_count": 0,
      "stale_signals": []
    },
    {
      "name": "interrupted_builds",
      "status": "healthy",
      "artifacts": []
    }
  ],
  "overall_status": "healthy"
}
```

overall_status: "healthy" if all checks healthy, "warning" if any warning, "error" if any error.
Use outputOK() for the result.
