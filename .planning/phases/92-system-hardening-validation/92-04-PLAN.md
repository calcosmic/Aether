Per D-10, create validation functions and comprehensive tests for all v1.13 file formats.

**1. Add `ValidateHeartbeatFile()` to `cmd/heartbeat_monitor.go`:**
```go
func ValidateHeartbeatFile(data []byte) (HeartbeatFile, error)
```
- Unmarshal JSON, check required fields: worker_id (non-empty), timestamp (valid RFC3339), phase (>0)
- Return actionable error messages: "heartbeat file: missing required field 'worker_id'" or "heartbeat file: invalid timestamp format, expected RFC3339 (e.g., 2026-05-02T14:30:00Z), got 'bad-timestamp'" or "heartbeat file: phase must be >= 0, got -1"

**2. Add `ValidateGateResults()` to `cmd/gate_results.go` (or create if not existing):**
```go
func ValidateGateResults(data []byte) error
```
- Unmarshal JSON, check: phase field exists and is positive, gates field is a map, each gate status is one of: "passed", "failed", "skipped", "not-reached"
- Return actionable errors: "gate-results.json: missing required field 'phase'" or "gate-results.json: invalid status 'unknown' for gate 'tests_gate', valid statuses: passed, failed, skipped, not-reached"

**3. Add `ValidateLearningEntry()` to `cmd/validation_v113.go` (new file if needed, or inline in test file):**
```go
func ValidateLearningEntry(data []byte) error
```
- Check: id (non-empty), phase (>0), content (non-empty), classification (one of: repo-local, hive-shareable, blocked, needs-user-approval), confidence (0.0-1.0), scope (non-empty), evidence has timestamp
- Return actionable errors for each violation

**4. Add `ValidateSkillFrontmatter()` (can reference existing skill parsing):**
```go
func ValidateSkillFrontmatter(content string) error
```
- Parse frontmatter from SKILL.md content
- Check: name (non-empty), category (colony or domain), roles (non-empty array)
- Return actionable errors

**5. Add `ValidateTrackedProcessJSON()` to `cmd/validation_v113.go`:**
```go
func ValidateTrackedProcessJSON(data []byte) error
```
- Unmarshal as `trackedProcessFile`, check each process: PID > 0, WorkerName non-empty, SpawnedAt is valid time
- Return actionable errors
